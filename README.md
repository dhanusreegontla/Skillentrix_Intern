# Skillentrix_Intern
## Project Title

**House Price Prediction Using Machine Learning**

## Project Description

This project predicts house prices using machine learning regression algorithms based on various house features such as area, number of bedrooms, bathrooms, garage capacity, living area, and overall quality. The project performs data preprocessing, handles missing values, applies feature scaling, trains multiple regression models, and predicts house prices with high accuracy. Among all models, the Gradient Boosting Regressor provides the best performance.

## Objectives

* Predict house prices accurately using machine learning techniques.
* Compare different regression algorithms.
* Handle missing values and preprocess data effectively.
* Evaluate model performance using R² Score and RMSE.

## Technologies Used

## Algorithms Used

* Linear Regression

## Features

* Dataset creation and preprocessing
* Missing value handling
* Feature scaling using StandardScaler
* Train-test split
* Hyperparameter tuning using GridSearchCV
* House price prediction
* Automatic generation of `submission.csv`

The system successfully predicts house prices and generates a `submission.csv` file containing the predicted prices. Gradient Boosting Regressor achieved the highest prediction accuracy among all the regression models tested.
